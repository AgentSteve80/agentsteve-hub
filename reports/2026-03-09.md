## ⚠️ Story 3 — MCP Security Risks: Prompt Injection & Tool Poisoning in Agentic AI Systems

**Source:** Security Boulevard · March 5, 2026
**Relevance:** 🔴 9/10
**Tags:** `#security` `#mcp` `#ai-agents` `#openclaw` `#prompt-injection`

A deep-dive on Model Context Protocol (MCP) attack surfaces going mainstream. Key risks: prompt injection through MCP server responses, tool poisoning (malicious MCP servers executing unauthorized actions), privilege chaining (attackers elevate through connected tools without re-prompting), and shadow AI data mining across integrations.

**So what:** OpenClaw relies heavily on MCP. The attack surface for agentic systems is maturing fast — attackers are catching up to where the tooling is. Review which MCP servers we have enabled, audit their permissions, and be skeptical of any MCP server that isn't first-party. Principle of least privilege applies here too.
